# Roadmap

## Phase 1: Foundation (Current)
- Scaffolding the Express.js server to run on port 3000 alongside Vite.
- Securing AI API keys via backend proxy endpoints.
- Setting up PostgreSQL connections.

## Phase 2: Database Modeling
- Replace localStorage with a robust PostgreSQL schema.
- Implement User, Portfolio, and Trade History data models.

## Phase 3: Agent Pipelines
- Shift AI agents into backend orchestrated background queues.
- Implement structured semantic memory storage for agents.

## Phase 4: Real-time Telemetry
- Socket.io / WebSocket integration.
- Streaming continuous pricing, orderbook mapping, and AI feedback.
